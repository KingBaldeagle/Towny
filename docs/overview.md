# Overview

NeoTowny is a NeoForge 1.21.1 mod that adds a **Resident → Town → Nation** hierarchy to Minecraft servers. It provides:

- Grid‑based land claiming (16×16 block chunks)
- Per‑plot ownership and sale mechanics
- Economy integration with **LightmansCurrency** (or a closed no‑op provider)
- Automatic tax collection for residents, towns and nations
- Outpost flag on town blocks for future strategic features
- HUD that shows the area a player is standing in
- Simple war system allowing nations to declare war on each other
- Placeholder service for other mods to query resident information

All functionality is accessed through the `/town`, `/nation`, `/plot`, `/resident` (or `/res`) command prefixes and a few utility commands.
