# Coin Mint
![](https://imgur.com/Yo2DID6.png)

The coin mint is used to craft coins out of their base materials, and melt coins back into their base materials.

![](https://i.imgur.com/BeAQL52.png)

To craft a coin you simply place a coins base material in the left slot (copper/iron/gold/netherite ingots, or emerald/diamond gems) and the minting process will begin automatically (v2.1.2.2 or newer, older versions require that you click on the arrow button to manually trigger the minting process).

The Coin Mint can have items inserted or extracted from any side. Inserted items will only go into the input slot, and items can only be extracted from the output slot.

**Note**: Default coin minting & melting recipes can be disabled by the server owner. Check JEI/REI to view which recipes are available

## Coin Mint Configuration

The Server Config has various options to enable/disable the default Coin Mint recipes. See the [Config Files](https://github.com/Lightman314/LightmansCurrency/wiki/Config-Files#common-config) section for full details.

## Coin Mint Recipes

The Coin Mint can have customized recipes added with a data pack. See the [Coin Mint Recipe Data](https://github.com/Lightman314/LightmansCurrency/wiki/Coin-Mint-Recipe-Data) page for details on how these recipes are formatted if you wish to add your own.
