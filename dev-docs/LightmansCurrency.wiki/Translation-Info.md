### Help by providing more translations
If you're interested in playing with this mod, but would like this to support another language you may go to the Issues page and create a "Translation Upload" issue and attach the relevant language json file in the requested field. You can find an up-to-date copy of the english language file [here](https://github.com/Lightman314/LightmansCurrency/blob/LC-1.20.1/src/generated/resources/assets/lightmanscurrency/lang/en_us.json), and if you require any additional context for any translation keys feel free to ask for help on my Discord Group

### Updating translations
As of v2.2.5.0, I've created a [Translation Changelog](https://github.com/Lightman314/LightmansCurrency/wiki/Translation-Changelog) wiki page that I update every so often that you can use to get recent changes if you've translated in a somewhat recent version, as well as getting previews of upcoming translation changes.

**Disclaimer**: _These translations are provided on a volunteer basis, and as an English speaking person, I cannot in any way validate their contents. Please report any inaccurate or inappropriate translations in the Issues tab of this github._
